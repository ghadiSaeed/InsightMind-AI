# InsightMind AI

An AI-powered Mental Health Research Assistant designed to help researchers analyze academic papers efficiently using Large Language Models (LLMs).

## Features

- Upload and manage multiple PDF research papers
- PDF Viewer with page navigation and text search
- AI-powered document question answering
- Research Dashboard
- Automatic topic extraction
- Keyword and research trend analysis
- Literature Review generation
- Research Gap identification
- Interactive Mind Map
- Presentation Generator
- Notes and Bookmarks
- Export Notes and Presentation

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
- PDF.js
- Gemini 2.5 Flash API
- Vis Network
- html2pdf.js
- Marked.js

## AI Model

This project uses **Google Gemini 2.5 Flash** as the Large Language Model (LLM) for:

- Research question answering
- Topic extraction
- Literature analysis
- Mind map generation
- Presentation generation
- Dashboard insights

## How to Run

1. Download the project.
2. Open `index.html` in your browser.
3. Enter your Gemini API Key.
4. Upload one or more research papers.
5. Start analyzing your documents.
